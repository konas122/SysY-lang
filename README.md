# SysY-lang

![status build](https://github.com/konas122/SysY-lang/actions/workflows/build.yml/badge.svg?branch=main)

SysY 语言编译器 (C subset)

基于 x86 Linux 的平台, 实现了编译器的 **前端** , **IR**, **优化**, **后端** 和 **链接**.
